Grading: Homework is graded based on effort, not correctness. However, there is no partial credit; you must show substantial effort on every problem to receive any points. This homework is out of 2 points.

Homework Questions


Problem 1: Fibonacci
Return the nth number in the fibonacci sequence using iteration. Google Fibonacci Sequence if you need a refresher on it.

Problem 2: Hailstone
Print the hailstone sequence starting at n and return its length. The hailstone sequence is as follows: if n is even return n/2. if n is odd return (n*3) + 1. if n is 1, return n.

Problem 3: Largest Factor
Return the largest factor of n*n-1 that is smaller than n.

Problem 4: Factorial
Given an integer n, return factorial n (n!).

Testing
