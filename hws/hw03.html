Homework 3: Getting Started
Due by Friday 3/23 11:59 PM PST

Instructions

Download hw03.zip.
Submission:
Grading: Homework is graded based on effort, not correctness. However, there is no partial credit; you must show substantial effort on every problem to receive any points. This homework is out of 2 points.

Homework Questions


Problem 1: G
Return the value of g(n), computed recursively.

Problem 2: Has Seven
Given a list of integers, return whether or not the list contains a 7 in it. Hint: remember you can use for loops to iterate through lists.

Problem 3: Is Palindrome
Given a number in the form of a string, return whether or not the number is a palindrome aka whether or not the number is the same if it's reversed.

Problem 4: Count Change
Once the machines take over, the denomination of every coin will be a power of 2: 1-cent, 2-cent, 4-cent, 8-cent, etc. There will be no limit to how much a coin can be worth.
A set of coins makes changes for n if the sum of the values of the coin is n. For example, the following sets make change for 7: 7 1-cent coins, 5 1-cent and 1 2-cent coins, 3 1-cent and 1 4-cent coins, 1 1-cent and 3 2-cent coins, 1 1-cent, 1 2-cent, and 1 4-cent coins. There are a total of 6 different ways.
Write a function count_change that takes a positive integer n and returns the number of ways to make change from using these coins of the future.

Testing

To run the tests, go on your terminal/command line, navigate to the hw folder and input python3 test_hw00.py. The test will then run and output your score.
