Homework 0: Getting Started
Due by Friday 1/10 11:59 PM PST

Instructions

Download hw00.zip.
Submission:
Grading: Homework is graded based on effort, not correctness. However, there is no partial credit; you must show substantial effort on every problem to receive any points. This homework is out of 2 points.

Homework Questions


Problem 1: Basic Arithmetic Functions
Fill in the missing code for the functions add(),sub(),mul() and div().

Problem 2: Quick Maths
Using the functions that you just implemented, fill in the missing blanks to solve the two following arithmetic equations quick_maths_one and quick_maths_two.

Problem 3: A Plus Abs B
Fill in the blanks in the following function for adding a to the absolute value of b, without calling abs. You may not modify any of the provided code other than the two blanks.

Problem 4: Two of Three
Write a function that takes three positive numbers and returns the sum of the squares of the two largest numbers. Use only a single line for the body of the function.

Problem 5: If Function
Write a function returns true_result if the condition is True and false_result if the condition is False.

Testing


Testing will be done locally, but keep in mind that you should not change or mess with the test.py files.
To run the tests, go on your terminal/command line, navigate to the hw folder and input python3 test_hw00.py. The test will then run and output your score.
