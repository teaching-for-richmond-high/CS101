Homework Questions


Problem 1: Fibonacci Recursive
Return the nth number of the fibonacci sequence using recursion!

Problem 2: Product
Return the product of the first n terms in a sequence where n is a positive integer and term is a function that takes one argument.

Problem 3: Repeated
Return the function that computes the nth application of f.

Problem 4: Triangle
We have a triangle made of blocks. The topmost row has 1 block, the next row down has 2 blocks, the next row has 3 blocks, and so on. Compute recursively(no loops or multiplication) the total number of blocks in such a triangle with the given number of rows.
